About

Are you tired of companies dumping toxic waste into places that should be kept clean? Greener Map is a place where you can take pictures of companies that's illegally harming the ecosystem and gives us the ability to report such problem to the right people. By doing so, we are able to create a future where companies are held accountable for the waste they create and lead to a Greener Map.

Goal

Our company wants to bring awareness to improper disposal of toxic waste and deforestation through the use Instagram's Geotag location API. Encouraging users to report incendents that the feel has not been treated properly.

Use the #greenerworld on Instagram to report

Application

×

Our product is based around having large amounts of user inputed images and data, we decided to use Instagram as a platform which makes it easy for users to interact with, as most people already use instagram. The only user action that is required to submit to our application is to post to instagram with the hashtag #GreenerWorld. We utilized a PhantomBuster API to scrape data from Instagram, which generates a CVS/Json file that has all information on posts that use the hashtag such as username, image link, and location. Using the CSV file we can input the location data of all posts into the Google Maps API to create custom maps of our collected data. With this we can see all locations in which user reported corporate abuse of the environment.
